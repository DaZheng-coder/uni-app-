# uni-app-
主要使用uni-app开发的在线笔记

## 2020.12.24 
  ### 初步完成了一下一些基本功能：
    1.首页(文件夹、笔记列表、添加删除文件夹、添加删除笔记、设置笔记是否可见), 
    2.个人页面(登录、注册、退出登录、更换头像、更换用户名), 
    3.分享页面，
    4.笔记页面(编辑、定时自动保存、富文本)

    5.考虑是否首页添加笔记可选标签
